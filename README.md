Life Shield is a web-based emergency response and personal safety application developed as my first semester project for the AICT Lab (Application of Information and Communication Technologies Lab). This project was designed to address real-world safety concerns by providing a fast, accessible, and technology-driven solution for individuals in emergency situations.
The primary objective of Life Shield is to ensure that users can quickly seek help when facing danger. The application includes an SOS emergency alert system that allows users to instantly trigger an alert with a single click. In a real-world implementation, this feature can be connected to trusted contacts and emergency services to share alerts and location details.

The system also includes:
-> Live Location Feature – Designed to help users share their real-time location during emergencies.
-> Trusted Contact Management – Users can add and manage emergency contacts for quick communication.
-> Emergency Helpline Section – Provides quick access to important emergency numbers.
-> Incident Reporting Module – Allows users to report unsafe situations or areas.
-> Basic User Interface Structure – Designed with simplicity and accessibility in mind for quick response.
